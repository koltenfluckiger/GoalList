# GoalieList
A goal tracking application that uses CoreData for persistent storage.

<br>

<h1>Features</h1>
- Core Data
<br>
- CAAnimation
<br>
- Keyboard frame animation
<br>
- Custom presenter and dismissing controllers
<br>
- Deleting goals
<br>
- Setting goal points
<br><br><br>
